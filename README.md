# デイトレード運用支援エージェント (Day Trade Operation Agent)

## 概要
本システムは、日次デイトレードにおける「銘柄選定（スクリーニング）」と「資金管理（ポジション計算）」を支援するためのPython製ダッシュボードツールです。
Streamlitを用いたWeb GUIで、日経225構成銘柄のスコアリングや、リスク許容度に基づいた適正株数の算出を自動化します。

**注意:** データ取得に `yfinance` を使用しているため、株価データには15〜20分程度の遅延が発生します。リアルタイムの発注判断ではなく、寄り付き前の分析や後場に向けた戦略立案、資金管理シミュレーションに使用することを想定しています。

## 機能一覧
1.  **銘柄スクリーニング**
    * 日経225銘柄（設定リスト）から、独自のスコア基準（出来高加速、ギャップ、VWAP位置など）に基づいて有望銘柄をランキング表示。
    * 上位3銘柄を「監視候補」としてカード表示。
2.  **チャート分析**
    * 1分足ローソク足チャート ＋ VWAP（出来高加重平均価格）の描画。
3.  **資金管理計算機**
    * エントリー価格と許容リスク額（例: 4,000円）を入力するだけで、「適正株数」「損切りライン(SL)」「利確目標(TP)」を自動算出。
    * デイトレードの損失をコントロールするための計算を瞬時に行います。

## 必要要件 (Requirements)
* Python 3.8 以上
* 以下のPythonライブラリ:
    * streamlit
    * yfinance
    * pandas
    * plotly

## インストール手順

1.  **リポジトリのクローンまたはダウンロード**
    フォルダを任意の場所に配置してください。

2.  **ライブラリのインストール**
    ターミナル（コマンドプロンプト）で以下のコマンドを実行します。
    ```bash
    pip install streamlit yfinance pandas plotly
    ```

## 使い方
